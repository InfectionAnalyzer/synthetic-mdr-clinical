Synthetic Clinical Data Toolkit

This toolkit enables generation of realistic, privacy-preserving synthetic clinical datasets for MDR modeling and stewardship research.

  • Generate data using GaussianCopula or CTGAN
  • Validate distributions, correlations, clustering (PCA, t-SNE)
  • Ready for Google Colab and offline use

Quick Start (Google Colab)

pip install -r requirements.txt
python generate_synthetic.py --input your_data.csv --output synthetic_data.csv --method gaussian
